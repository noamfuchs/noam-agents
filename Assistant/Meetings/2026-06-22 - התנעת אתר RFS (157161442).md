## Key Takeaways

  - [**MVP Scope Defined:** The initial product will be a web app with core features only: user onboarding, group management, and a directory for contacting CEOs. Mentoring features are explicitly out of scope.](https://fathom.video/share/QyfNeGPQSYh5qmo9Wzo5WhTAq1ySxeiB?tab=summary&timestamp=986.0)
  - [**Tech Stack Decided:** A backend/frontend separation will be used to enable a future native app without a full rebuild. The developers will use a monorepo for the server and UI.](https://fathom.video/share/QyfNeGPQSYh5qmo9Wzo5WhTAq1ySxeiB?tab=summary&timestamp=1100.0)
  - [**Process Established:** Development is blocked until Yonatan & Noam deliver user stories and wireframes. The team will work asynchronously, with quick weekly syncs to maintain alignment.](https://fathom.video/share/QyfNeGPQSYh5qmo9Wzo5WhTAq1ySxeiB?tab=summary&timestamp=986.0)
  - [**Pilot Program Rejected:** A pilot using Noam's existing prototype was rejected. The team will focus on building a robust MVP from scratch to avoid technical debt and ensure a scalable foundation.](https://fathom.video/share/QyfNeGPQSYh5qmo9Wzo5WhTAq1ySxeiB?tab=summary&timestamp=1680.0)

## Topics

### Project Goals & Philosophy

  - [**Primary Goal:** Build a robust, scalable product with a solid backend and correct logic.](https://fathom.video/share/QyfNeGPQSYh5qmo9Wzo5WhTAq1ySxeiB?tab=summary&timestamp=922.0)
      - [**Rationale:** Avoid repeating past failures where projects were built twice due to poor foundations.](https://fathom.video/share/QyfNeGPQSYh5qmo9Wzo5WhTAq1ySxeiB?tab=summary&timestamp=922.0)
  - [**MVP Focus:** Deliver maximum impact for end-users (runners, CEOs) in the shortest time.](https://fathom.video/share/QyfNeGPQSYh5qmo9Wzo5WhTAq1ySxeiB?tab=summary&timestamp=1160.0)
      - [**Rationale:** The UI for internal roles (e.g., group managers) will be simple and functional, not polished.](https://fathom.video/share/QyfNeGPQSYh5qmo9Wzo5WhTAq1ySxeiB?tab=summary&timestamp=1160.0)
  - [**Team Structure:** A small, focused team will work asynchronously to minimize friction.](https://fathom.video/share/QyfNeGPQSYh5qmo9Wzo5WhTAq1ySxeiB?tab=summary&timestamp=573.0)

      - [CEO directory & contact functionality](https://fathom.video/share/QyfNeGPQSYh5qmo9Wzo5WhTAq1ySxeiB?tab=summary&timestamp=986.0)
  - [**Out-of-Scope Features:**](https://fathom.video/share/QyfNeGPQSYh5qmo9Wzo5WhTAq1ySxeiB?tab=summary&timestamp=1027.0)
      - [Mentoring features](https://fathom.video/share/QyfNeGPQSYh5qmo9Wzo5WhTAq1ySxeiB?tab=summary&timestamp=1027.0)
  - [**User Stories:** Yonatan & Noam will write user stories to define the MVP's requirements.](https://fathom.video/share/QyfNeGPQSYh5qmo9Wzo5WhTAq1ySxeiB?tab=summary&timestamp=1061.0)
      - [**Format:** "As a \[user role\], I want to \[action\] so that I can \[goal\]."](https://fathom.video/share/QyfNeGPQSYh5qmo9Wzo5WhTAq1ySxeiB?tab=summary&timestamp=1890.0)
      - [**Rationale:** This text-based format is the required input for the developers to begin building the data models (e.g., how a user or event is structured).](https://fathom.video/share/QyfNeGPQSYh5qmo9Wzo5WhTAq1ySxeiB?tab=summary&timestamp=1872.0)

### Technical Approach

  - [**Architecture:** Backend/frontend separation.](https://fathom.video/share/QyfNeGPQSYh5qmo9Wzo5WhTAq1ySxeiB?tab=summary&timestamp=1100.0)
      - [**Rationale:** This allows the team to replace the web UI with a native app in the future without rebuilding the core logic.](https://fathom.video/share/QyfNeGPQSYh5qmo9Wzo5WhTAq1ySxeiB?tab=summary&timestamp=1140.0)
  - [**Repository:** A monorepo will be used for the server and UI.](https://fathom.video/share/QyfNeGPQSYh5qmo9Wzo5WhTAq1ySxeiB?tab=summary&timestamp=1200.0)
  - [**Pilot Program Rejected:** A proposal to use Noam's existing Vercel prototype for a pilot was rejected.](https://fathom.video/share/QyfNeGPQSYh5qmo9Wzo5WhTAq1ySxeiB?tab=summary&timestamp=1680.0)
      - [**Rationale:** The prototype's backend logic is unknown and its UX is not polished enough for the core CEO connection feature. The team will build the MVP from scratch to ensure a clean, scalable foundation.](https://fathom.video/share/QyfNeGPQSYh5qmo9Wzo5WhTAq1ySxeiB?tab=summary&timestamp=1680.0)
