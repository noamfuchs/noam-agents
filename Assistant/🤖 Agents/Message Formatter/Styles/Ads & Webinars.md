# Style Profile — Ads & Webinars (פרסום וובינרים)

## Audience
Prospects and registered leads — people who signed up for a webinar or are in a marketing WhatsApp group. They don't know you personally yet. This is marketing copy.

---

## Tone
- Exciting, professional, confident
- Creates anticipation and FOMO (fear of missing out)
- Speaks in "we" — the business speaks
- Feels valuable, not pushy
- Bold on key phrases to draw the eye
- Uses urgency when relevant (deadlines, limited spots)

---

## Two Message Types

### Type 1 — Webinar Announcement / Reminder
Use for: group welcome, day-before reminder, day-of reminder

**Pattern:**
```
*היי חברים, [context]! 👋*
*[date/time line] 📅*
[Hook line — "we're going deep into..." / "no filters, no theory..."]
בוובינר נדבר על:
🔹 *[topic]:* [1-line description]
🔹 *[topic]:* [1-line description]
...
הולך להיות *פרקטי*, *בגובה העיניים* ועם *המון ערך* 🚀
*שימו לב:*
📌 [Important note 1]
📌 [Important note 2]
[Registration/Zoom link if available]
*נתראה ב-[time]!* ✅
```

### Type 2 — Post-Webinar CTA / Follow-up
Use for: sending the booking link, limited time offer, personal outreach

**Pattern:**
```
[Brief context — "we got a lot of inquiries" / "for those who couldn't join..."]
[What's being offered — personal call, discount, etc.]
[What happens in the call/next step]
[Urgency — deadline, limited slots]
[Link]
```

---

## Formatting Rules
- **Bold** (*) on: event name, key value points, times, calls-to-action
- 🔹 for content/value bullet points inside the webinar description
- 📌 for important logistics notes
- ✅ at the end to confirm/close
- ⏳ for urgency / deadlines
- 🎁 for bonuses or special offers
- Zoom link goes at end, never buries the hook
- WhatsApp bold with * not markdown

---

## Emoji Bank
👋 — greeting / welcome
📅 — date/time announcement
🔹 — content bullet
💰 — financial/leverage point
🔧 — renovation/operations point
🚀 — exciting closer / launch feel
📌 — important logistics
✅ — confirmation / done
⏳ — urgency / deadline
🎁 — bonus / special offer
📞 — call / personal session

## Real Examples

**Example 1 — Day-before announcement:**
*היי חברים, ברוכים הבאים לקבוצת הוובינר! 👋*
*מחר זה קורה: יום שני (6.4) | 20:00 | נפגשים בזום 📅*
אנחנו הולכים לצלול עמוק לתוך "העולם האמיתי" של ההשקעות – בלי פילטרים ובלי תיאוריות מיותרות. בוובינר נדבר על:
🔹 *למה דווקא עכשיו?* היתרונות של השוק האמריקאי ב-2026.
🔹 *ממשקיע ליזם:* איך הופכים רכישה בודדת לעסק עם הכנסה דולרית קבועה.
🔹 *הכוח של המינוף:* איך עובדים עם מימון גבוה (עד 90%!) למשקיעים זרים. 💰
🔹 *ניהול בשלט רחוק:* איך משתלטים על שיפוצים וצוותים בשטח בלי לעלות על מטוס. 🔧
🔹 *בלי פילטרים:* מה הסיכונים, למי זה לא מתאים ואיך נראה יום בחיי יזם.
הולך להיות *פרקטי*, *בגובה העיניים* ועם *המון ערך* שיכול לפתוח לכם כיוון חדש לגמרי. 🚀
*שימו לב:*
📌 מומלץ לפנות את הזמן ולהגיע מרוכזים עם מחברת.
📌 הקישור לזום יישלח כאן בקבוצה בסמוך למועד השידור.
*נתראה מחר ב-20:00!* ✅

**Example 2 — Day-of:**
*היי חברים, ברוכים הבאים לקבוצת הוובינר! 👋*
*היום זה קורה! | 20:00 | נפגשים בזום 📅*
[same content bullets]
*נתראה ב-20:00!* ✅
קישור לזום ישלח בסמוך לאירוע ✅

**Example 3 — Post-webinar CTA:**
קיבלנו המון פניות, אבל חשוב לנו לא לפספס חבר'ה שלא הצליחו לעלות לשידור בגלל המצב.
לכן, פתחנו גם כאן את האפשרות לשיחת התאמה וייעוץ אישית 📞
מה יקרה בשיחה?
נבדוק יחד אם התוכנית מדויקת ליעדים שלכם, ואם תתאימו – תוכלו לממש הטבה בלעדית של 1,500 ש"ח להרשמה! 🎁
שימו לב: הקישור וההטבה פעילים עד חצות (00:00) בלבד! ⏳
תיכנסו לרשימה כאן:
[link]

---

## Output Rules
- Output the message in Hebrew only
- No explanation, just the ready-to-send message
- If the raw input mentions a date and topic → use Webinar Announcement format
- If the raw input mentions follow-up, booking call, or discount → use Post-Webinar CTA format
- Keep all factual details (times, dates, links, names) exactly as given
- Bold key phrases, not full sentences
