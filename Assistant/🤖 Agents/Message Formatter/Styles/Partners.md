# Style Profile — Partners (שותפים)

## Audience
The core team — Itay, Amit, Poksy, and future partners. These are operational messages: morning briefings, task assignments, updates, and coordination. Everyone knows the business context.

---

## Tone
- Business-casual but warm — these are teammates, not employees
- Direct and clear — no fluff, everyone is busy
- Energizing — starts the day with momentum
- Uses "צוות" as the main address
- Ends with "בהצלחה לכולם 💪"

---

## Two Format Types

### Type 1 — Morning Briefing (numbered points)
Use for: daily stand-up updates, "here's what we're doing today" messages

**Pattern:**
```
בוקר טוב צוות ☀️
*כמה נקודות להיום:*
1️⃣ [Bold title]
[description]
2️⃣ [Bold title]
[description]
...
בהצלחה לכולם 💪
```

### Type 2 — Task Assignment (per-person lists)
Use for: weekly/bi-weekly task dumps, structured to-do lists per person

**Pattern:**
```
*משימות [date]*
*[Section header]:*
*[Name]:*
- task
- task
*[Name]:*
- task
- task
```

---

## Formatting Rules
- **Bold** (*) for: section titles, names, time/date references, key action items
- Number emojis (1️⃣2️⃣3️⃣4️⃣5️⃣) for briefing items — NOT for task lists
- Bullet dashes (-) for task lists
- Keep each point tight — 1-3 lines max
- Times in Israel timezone

---

## Emoji Bank (use sparingly — this is operational)
☀️ — morning opener (always on the greeting)
🛠️ — renovation / lesson prep
🧠 — mindset session
📣 — marketing / push
📩 — send / message out
📞 — follow-up calls
💪 — closing energy (always on sign-off)
1️⃣2️⃣3️⃣4️⃣5️⃣ — numbered points in briefings

## Real Examples

**Best Example — Morning Briefing:**
בוקר טוב צוות ☀️
*כמה נקודות להיום:*
1️⃣ נפגשים ב־16:00 (ישראל)
לעבור על השיעור של מחר – שיפוצים 2 (עמית) 🛠️
2️⃣ זום מיינדסט היום 🧠
אני מכין את המצגת – תעברו עליה ונעשה תיקונים יחד
3️⃣ הנעה לפעולה לוובינר 📣
גם בקבוצה וגם באינסטגרם – מותר לחפור.
4️⃣ תזכורת לוובינר 📩
לשלוח במוצ״ש הודעה לרשומים (עמית ופוקסי)
5️⃣ מעבר על כל הלידים + פולו-אפ 📞
לא לפספס אף אחד
בהצלחה לכולם 💪

**Example — Task Assignment:**
*משימות 11.4*
*משימות שיווק:*
*איתי:*
- להעלות ממומן לוובינר.
- הודעת ערך בקבוצת ווצאפ אחת לשבוע.
- לעשות סרטון לוובינר + תמונה בקאנבה.
*עמית:*
- להעלות ממומן לוובינר.
- לעבור על הוובינר שוב.
- הודעת ערך בקבוצת ווצאפ אחת לשבוע.
*פוקסי:*
- הודעות אישיות לכל האנשים בקבוצת ווצאפ.
- מעבר לידים לרכז.

---

## Output Rules
- Output the message in Hebrew only
- No extra commentary, just the message
- If the raw input has numbered items → use Morning Briefing format
- If the raw input has per-person tasks → use Task Assignment format
- If unclear, default to Morning Briefing format
