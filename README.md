# Noam's Second Brain

The vault root is **your** clean note-taker. Two areas:

- **`📓 Journal/`** — your own writing. Yours only. The assistant reads it for context but never clutters it.
- **`Assistant/`** — everything the assistant uses (memory, people, projects, meetings, system, automations). Ignore it whenever you want; open it if you're curious.

Tasks / status / goals live in **Notion** (the daily front-of-house), not here — see `Assistant/system/notion-command-center.md`.

Open it via:
```
cd ~/second-brain && claude
```
Canonical location: `~/Dropbox/MY BRAIN/` (Dropbox-synced). `~/second-brain` and `~/MY BRAIN` are symlinks to it. A separate abandoned copy at `~/Desktop/MY BRAIN` is NOT this vault — ignore it.
